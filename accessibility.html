Our Commitment

We are committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience for everyone and apply the relevant accessibility standards.

Accessibility Features

  • Keyboard Navigation: All features are accessible using keyboard controls
  • Screen Reader Compatible: Our content is properly structured for screen readers
  • Text Resizing: Content can be resized up to 200% without loss of functionality
  • Color Contrast: We maintain WCAG 2.1 AA standard contrast ratios
  • Alt Text: All images include descriptive alternative text
